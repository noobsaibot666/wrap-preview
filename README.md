# CineFlow Suite — The Production & Post Hub

**CineFlow Suite** is a professional macOS application designed for film and video production. It allows users to quickly generate high-resolution contact sheets (PDF/PNG), manage shot lists, and review footage with precision.

## Key Features

- **Local-First Processing**: Fast footage scanning and metadata extraction without cloud dependencies.
- **Shot Planner**: Manage pre-production references and gear lists.
- **Safe Copy**: Verified file ingest with checksum validation.
- **Review Core**: On-set review with frame notes, comments, and annotations.
- **Dynamic Exports**: Professional exports for FCPXML and Director Packs.

## Project Structure

- `src/`: Frontend React application.
- `src-tauri/`: Rust core logic and system integrations.
- `docs/`: Comprehensive project documentation.
  - `docs/appstore/`: App Store submission and privacy configurations.
  - `docs/reports/`: Functional hardening and certification reports.
